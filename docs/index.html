Intelligent Resume Optimizer

JD-aware resume tailoring with FastAPI, Streamlit, ATS scoring, structured resume retrieval, export generation, CI on GitHub Actions, and optional Telegram notifications.

Open Repository View CI Runs

Run Locally

Install dependencies and start the app with streamlit run streamlit_app.py.

Deploy the App

Use Streamlit Community Cloud for the simplest live deployment, or deploy the included Docker image on a container host.

Notifications

Add TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID as GitHub secrets to receive CI updates in Telegram.

GitHub Pages can host this landing page, but not the interactive Streamlit or FastAPI runtime itself. For the live app, deploy from this repository to Streamlit Cloud or a Docker host.
